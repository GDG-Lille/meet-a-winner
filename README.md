## How to build for production ?

Nothing to do, [Circle CI](https://circleci.com/gh/GDG-Lille) does it :) (@see .circleci/config.yml)

## How to run in local ?

* Run `npm install` to install all dependancies.
* Run `npm run serve` to deploy a dev server.
